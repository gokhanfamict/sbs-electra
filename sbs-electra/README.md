# SBS Electra Website

Professionele website voor SBS Electra - elektrotechnische installaties.

## Installatie

```bash
cd sbs-electra
npm install
npm run dev
```

## Deploy naar Vercel

1. Push naar GitHub
2. Import in Vercel
3. Deploy automatisch

## Technologie

- Next.js 14
- Tailwind CSS
- Framer Motion
- Lucide Icons
